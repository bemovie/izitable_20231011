# 이지테이블(IZITable)

실시간 음식점 예약 서비스

## 제작자

학과 : 스마트소프트웨어

소그룹 : 비타민 C팀

성명 : 정성호, 정승복, 윤진수, 심소영


## 제작목적

코로나바이러스의 위기 단계가 점차 하향됨에 움츠러들었던 외식 인구가 다시 정상화되어감에 따라 실시간 음식점 예약 플랫폼을 제공하면 좋겠다고 생각함.

## 핵심내용(활용효과)

- 마스크 필수 착용 해제 이후 늘어난 외식 문화에 알맞은 서비스
 
- 인기 음식점 카테고리 제공

- 소비자의 니즈에 맞는 검색 기능 제공

- 간편한 검색 서비스에 따른 예약 서비스로 업그레이드된 편의성 제공
